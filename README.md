# C++编程模版

请大家按照模版进行c++的编程，并准备好

## 配置

1. 请使用vscode 进行开发工作 并安装如下插件
    1. cmake & cmake tools
    2. Doxygen Documentation Generator
    3. C/C++ 
    4. clang-format
    5. cpplint


## 使用

1. 各模块写于modules中
    1. 打包为类
    2. 使用_test.cc进行功能测试
    3. 若测试需要数据，请创建data文件夹并在其中读取
2. cmakelists 请仿照建议用法进行使用
3. 各类请使用Doxygen插件 进行代码内 程序使用说明
4. 请遵循 模块依赖关系进行使用
